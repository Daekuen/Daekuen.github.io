# 팩토리얼

### Section.8 - 11

---

#### My Solution

```javascript
function solution(n) {
  let answer;
  function DFS(n) {
    if (n === 1) return 1;
    else return n * DFS(n - 1);
  }
  answer = DFS(n);
  return answer;
}

console.log(solution(5));
```

---

#### Summary

- 재귀함수가 값을 리턴받는 형태.
- DFS함수에서 값을 리턴하면, 리턴값이 answer에 쌓이게 된다.
- 결과적으로 answer = 5x4x3x2x1가 된다.
